# retail-sales-sql-analysis
Retail sales database analysis using SQL to answer business questions, generate insights, and demonstrate practical SQL skills for a data analytics portfolio.


## Project Overview

This project analyzes sales transaction data using SQL to answer real business questions. The analysis demonstrates how SQL can be used to extract meaningful insights, identify sales trends, and support business decision-making.

The project includes SQL queries, business interpretations, recommendations, and a final analytical report.

## Objectives

The objectives of this project are to:

- Analyze sales transactions using SQL.
- Practice writing SQL queries to answer business questions.
- Generate business insights from sales data.
- Improve SQL skills for data analytics.
- Build a professional data analytics portfolio project.

## Dataset

The dataset contains sample sales transaction records with the following fields:

- Order ID
- Product
- Quantity
- Unit Price
- Customer City

The dataset was created for learning and portfolio purposes.

## Business Questions Answered

The analysis answered several business questions, including:

- How many sales transactions are recorded?
- Which product generated the highest revenue?
- Which product sold the most units?
- Which transactions involved high sales quantities?
- Which products matched specific search patterns?
- What are the top-performing sales transactions?
- Which cities recorded specific sales activities?

## Key Findings
The analysis revealed several important insights:

- Cooking Oil 1L generated the highest total revenue.
- Sugar 1kg recorded the highest number of units sold.
- Most sales transactions involved quantities between 15 and 25 units.
- Freetown recorded multiple sales transactions, including the largest order from that city.
- SQL queries helped identify sales trends, top-performing products, and customer purchasing patterns.

## Business Recommendations

Based on the analysis, the following recommendations are suggested:

- Increase inventory levels for Sugar 1kg due to its high sales volume.
- Continue promoting Cooking Oil 1L since it contributes the highest revenue.
- Monitor medium-sized orders (15–25 units) to maintain adequate stock levels.
- Use SQL reporting regularly to support inventory and sales decision-making.
